# Loan-Approval-Prediction
Loan Approval Prediction System

This project predicts whether a loan will be approved or rejected based on applicant details using Machine Learning classification models.
The system is deployed using FastAPI and Streamlit.

Problem Type

Classification

Target column: Loan_Status

Models Used

Logistic Regression

K-Nearest Neighbors (KNN)

Random Forest Classifier

Tech Stack

Python

Pandas, NumPy

Scikit-learn

FastAPI

Streamlit

Pickle

Output

✅ Loan Approved

❌ Loan Rejected
