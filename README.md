# Loan-Approval-Prediction
Loan Approval Prediction System

This project predicts whether a loan will be approved or rejected based on applicant details using Machine Learning classification models.
The system is deployed using FastAPI and Streamlit.

Problem Type

Classification

Target column: Loan_Status

1 → Loan Approved

0 → Loan Rejected

Models Used

Logistic Regression

K-Nearest Neighbors (KNN)

Random Forest Classifier

Tech Stack

Python

Pandas, NumPy

Scikit-learn

FastAPI

Streamlit

Pickle

Features Used

Gender

Married

Dependents

Education

Self Employed

Applicant Income

Coapplicant Income

Loan Amount

Loan Amount Term

Credit History

Property Area


Output

✅ Loan Approved

❌ Loan Rejected
